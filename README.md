# 🍕 Pizza Sales Data Analytics Project

This project focuses on analyzing pizza sales data using **🗄️ SQL** and **📊 Power BI** to generate meaningful business insights. The project includes KPI analysis, sales trends, revenue analysis, and customer ordering patterns to help understand business performance and improve decision-making.

Using SQL queries, various insights such as total revenue, average order value, top-selling pizzas, monthly sales trends, and category-wise performance were analyzed. Finally, an interactive **📈 Power BI dashboard** was created to visualize the results effectively.

# ⚙️ Project Workflow

## 1️⃣ Data Collection & Understanding

The pizza sales dataset was imported and examined to understand:

* 📋 Order details
* 🍕 Pizza categories
* 💰 Revenue information
* 📅 Sales dates and trends

---

## 2️⃣ KPI Analysis Using SQL

SQL queries were used to calculate important business KPIs such as:

* 💵 Total Revenue
* 🛒 Total Orders
* 🍕 Total Pizzas Sold
* 📊 Average Order Value
* 📈 Average Pizzas Per Order

---

## 3️⃣ Sales Trend Analysis

Sales trends were analyzed to identify business performance over time.

### 📌 Analysis Includes:

* 📅 Daily Order Trends
* 🗓️ Monthly Sales Trends
* 📊 Category-wise Sales
* 📈 Pizza Size Performance

---

## 4️⃣ Best & Worst Selling Pizzas

SQL queries were used to identify:

* 🏆 Top 5 Pizzas by Revenue
* 📈 Top 5 Pizzas by Quantity Sold
* 📋 Top 5 Pizzas by Orders
* 📉 Bottom Selling Pizzas

This analysis helps understand customer preferences and product performance.
